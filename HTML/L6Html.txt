âœ… for Attribute (Used in <label>)  
- The for attribute links a label to an input field using the same id value.  
- When the user clicks the label, it automatically focuses on the input field.  

âœ… id Attribute (Used in <input>)  
- The id attribute uniquely identifies an input field.  
- It must match the for attribute in <label>.

html

    <label for="username">Enter Name:</label>
    <input type="text" id="username">
</form>
