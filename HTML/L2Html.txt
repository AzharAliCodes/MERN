 1. Comments in HTML  
html
<!-- This is a comment -->

- Comments in HTML start with <!-- and end with -->.  
- They are not displayed in the browser but are useful for documentation.

 2. HTML Headings  
HTML provides six levels of headings, from <h1> (largest) to <h6> (smallest).  

html
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>


ðŸ”¹ Usage: Headings are used to define titles and subtitles in web pages.  
ðŸ”¹ SEO Tip: <h1> is the most important heading and should be used for the main page title.

 3. Paragraph Tag  
html
<p>This is a paragraph.</p>

- The <p> tag is used to define paragraphs.  
- Browsers automatically add some space (margin) before and after paragraphs.  

---

 4. Types of Tags in HTML  

 (1) Structure Tags  
Used to define the structure of an HTML document.  
Examples:  
html
<html>, <head>, <body>


 (2) Semantic Tags  
These tags provide meaning to the content.  
Examples:  
html
<article>, <title>, <main>, <nav>, <footer>, <section>, <aside>

ðŸ”¹ Why use Semantic Tags?  
- They improve SEO.  
- They make the code more readable and accessible.  

 (3) Formatting Tags  
Used for styling text.  
Examples:  
html
<b>Bold</b>
<i>Italic</i>
<mark>Highlighted</mark>
<s>Strikethrough</s>

---

 Additional Information  

 Difference Between <b> and <strong>
- <b> makes text bold but has no extra meaning.  
- <strong> makes text bold and indicates importance.  

 Difference Between <i> and <em>
- <i> makes text italic for styling.  
- <em> makes text italic but also gives emphasis (important for screen readers).
