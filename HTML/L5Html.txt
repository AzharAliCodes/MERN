 1. Styling a Table using <style>  
The <style> tag is used to write CSS, which can be applied to tables for styling.  
Example:  
html
<style>
    table, th, td {
        border: 1px solid black;
        border-collapse: collapse;
    }
    th {
        background-color: lightgray;
    }
</style>

Here, we applied a border to the table, merged borders, and gave a background color to <th>.

- <thead> â†’ Defines the table header

 5. Column Grouping using <colgroup>  
The <colgroup> tag is used to style multiple columns together.  

 Example: Using <colgroup> for Styling  
html
<table>
    <colgroup>
        <col span="1" style="background-color: aqua;">
        <col span="2" style="background-color: lightgray;">
    </colgroup>
    <tr>
        <th>Name</th>
        <th>Age</th>
        <th>Country</th>
    </tr>
    <tr>
        <td>Ali</td>
        <td>22</td>
        <td>India</td>
    </tr>
    <tr>
        <td>Zoya</td>
        <td>24</td>
        <td>UK</td>
    </tr>
</table>

Here, the first column has a blue background, and the next two columns have a gray background.
